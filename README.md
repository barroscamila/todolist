## 🖥️ Projeto
Esse é um projeto web de uma "To do list", onde é possível adicionar, deletar e finalizar uma nova tarefa. 

## 🚀 Tecnologias
Esse projeto foi desenvolvido utilizando as seguintes tecnologias:

- HTML
- CSS
- JavaScript
- Git e Github
